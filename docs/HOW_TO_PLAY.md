## Understanding the System

### How Quests Work

1. **Automatic Tracking**
   - Every action you perform is recorded

   - Completion detected automatically

2. **Progress Updates**
   - Progress bars update in real-time
   - Stats refresh after each action
   - No manual claiming needed

3. **NFT Rewards**
   - Badge NFTs minted automatically
   - Stored on Hedera blockchain

Every action creates:

1. **Receipt NFT**

   - Proof of achievement
   - Contains quest details
   - Rarity indicator

3. **On-chain Record**
   - Permanent blockchain record
   - Viewable on HashScan
   - Cannot be altered

## Troubleshooting

### Quest Not Completing?

**Check Requirements:**
- Did you meet the minimum amount?
- Did you complete enough actions?
- Is the quest already completed?

**View Console:**
- Press F12 to open developer tools
- Check for error messages
- Look for completion logs

### Transaction Failing?

**Common Issues:**
- Insufficient HBAR in account
- Invalid private key
- Network connection issues

**Solutions:**
- Get testnet HBAR from portal.hedera.com
- Verify your private key is correct
- Check internet connection

### Progress Not Saving?

**Check:**
- Browser localStorage enabled
- Not in incognito mode
- Same account ID used

**Fix:**
- Enable localStorage in browser settings
- Use regular browser window
- Reconnect with same credentials
